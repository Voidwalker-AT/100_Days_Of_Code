## 🛠️ How to Run

<details>
<summary><b>⚡ C++ (GCC / G++)</b></summary>

```bash
# Navigate to the day's directory
cd Day1

# Compile
g++ Q1.cpp -o Q1

# Run
./Q1
```
</details>

<details>
<summary><b>☕ Java (JDK)</b></summary>

```bash
# Navigate to the day's directory
cd Day1

# Compile
javac Q2.java

# Run (class name depends on main method — e.g. Solution)
java Solution
```
</details>
