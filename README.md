## 🛠️ Execution Guides

### C++ (using GCC/G++)
To compile and run C++ files:
```bash
# Navigate to the day's directory
cd Day1

# Compile
g++ Q1.cpp -o Q1

# Run
./Q1
```

### Java (using JDK)
To compile and run Java files:
1. Ensure you have the class structure matching the entry point.
2. Compile and run:
```bash
# Navigate to the day's directory
cd Day1

# Compile
javac Q2.java

# Run the compiled class (e.g. Solution or Q2 depending on main method)
java Solution
```
